# Support

Use GitHub Issues for bugs, false positives, false negatives, and rule requests:

https://github.com/vigneshakaviki/mcp-check/issues

For security vulnerabilities in `mcp-check` itself, follow `SECURITY.md`.

When reporting a scanner issue, include:

- The MCP config snippet or a minimized reproduction.
- The command you ran.
- The expected finding.
- The actual finding.
- Whether the config is JSON, YAML, or TOML.
